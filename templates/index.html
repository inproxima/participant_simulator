AI Participant Simulator

To use the AI participant simulator, you first need to input your OpenAI API key. If you're not sure how get your key click here. Then, think about the purpose or function of the AI persona. Who is it that you would like to interview? This will help determine the personality and characteristics of the persona. Then write a description of the persona. Think about age, gender, occupation, ancestry, personality traits, and interests. Write a short description using clear and concise language and avoid technical jargon or complex sentences. Keep the tone and voice consistent throughout the description. Finally, review and revise the description to ensure that it accurately reflects the AI persona you have created. Once you are satisfied, click on GO!
